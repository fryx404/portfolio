## 🎯 作品の追加・編集方法

### 1. 作品データの編集
`scripts/works.js` を開いて、`WORKS_DATA` 配列に新しい作品を追加：

```javascript
{
    id: "unique-id",
    title: "作品名",
    description: "作品の哲学的記述...",
    technologies: ["使用技術1", "使用技術2"],
    features: [
        "特徴1",
        "特徴2", 
        "特徴3"
    ],
    duration: "開発期間",
    type: "作品タイプ",
    philosophy: "作品に込めた哲学...",
    status: "完成|開発中|構想中",
    year: 2024,
    image: "画像URL（Unsplash推奨）"
}
```

### 2. プロフィール情報の編集
`about.html` で以下の情報を直接編集：
- プロフィール名・肩書き
- 自己紹介文
- 創造の軌跡（タイムライン）
- SNSリンク

### 3. 画像の配置
- プロフィール画像: `images/profile/` に配置
- 作品画像: Unsplash等の外部URLを推奨

## 🔧 カスタマイズ方法

### テーマ色の変更
各CSSファイルの `:root` セクションで色変数を編集

### アニメーション調整
JavaScript ファイル内の以下パラメータを調整：
- パーティクル数: `maxParticles`
- アニメーション速度: `animationDelay`
- タイミング関数: CSS `transition-timing-function`

### 哲学的メッセージの編集
各HTMLファイルの `.philosophy-text` クラス要素を編集
